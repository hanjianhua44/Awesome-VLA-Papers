# Daily arXiv Feed

Automated daily paper digest from **arXiv cs.CV + cs.RO**, filtered for VLA / autonomous driving / robotics / embodied AI research from top institutions.

> Back to [Main Paper List](../README.md) | See [Workflow & Methodology](../WORKFLOW.md)

## How It Works

1. **Fetch** — Pull latest papers from arXiv (cs.CV + cs.RO)
2. **Filter** — Keyword relevance scoring (VLA, autonomous driving, manipulation, etc.)
3. **Identify** — Extract institution info from PDF first pages
4. **Select** — Keep only papers from [TIER1 institutions](../WORKFLOW.md#tier1-institution-list) (top labs, companies, universities)
5. **Report** — Generate a categorized Markdown digest

Reports run at 09:30 AM (local) on Tue–Sat, covering the previous day's arXiv listings (Tue report covers the Sat–Mon weekend batch).

Papers from the daily feed can be promoted to the [curated main list](../README.md) after manual review.
